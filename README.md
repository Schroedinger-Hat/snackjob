# Backend API

Available at https://snackjob.herokuapp.com or http://snackjob-api.schrodinger-hat.it

You can test it out as follow:

`
curl -X POST http://snackjob-api.schrodinger-hat.it/api/v1/snackjob -d 'name=ciao'
`
